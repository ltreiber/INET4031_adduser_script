# INET4031_adduser_script
Purpose: This program is designed to automate creating users 
and adding them to groups in Linux. The program is created with 
Python and takes an input file to run. 


Usage: To use this program, python3 must be installed and should
be run as root to avoid any issues. 

sudo python3 create-users.py < create-users.input

OR

cat create-users.input | sudo python3 create-users.py 

Have fun!
